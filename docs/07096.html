SYSTICK_TimerPeriodSet Function

This function is used to update SysTick Load value

C
void SYSTICK_TimerPeriodSet(
    uint32_t period
);
Preconditions

Systick should be stopped prior to setting up the new Load value by calling SYSTICK_TimerStop. This will make sure that the new value is used for the next Tick

Parameters
Parameters 
Description 
Period 
The period is specified as a number of clock ticks between SysTick interrupt. It must be between 1 and 16,777,216, inclusive. 
Returns

None.

Example
    SYSTICK_TimerStop();
    SYSTICK_TimerPeriodSet(0x00004567);
    SYSTICK_TimerStart();
