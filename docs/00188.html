ADC_ConversionStart Function

This function starts the ADC conversion. It needs to be called in application if conversion trigger source is not setup.

C
void ADC_ConversionStart();
Preconditions
  • ADC_Initialize() must have been called.
  • Automatic conversion is disabled.
Returns

None.

Remarks

None.

Example
    ADC_ConversionStart();
