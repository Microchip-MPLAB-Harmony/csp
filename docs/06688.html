USARTx_PARITY Enumeration

USART PARITY 

This may be used to check the type of parity with the USART peripheral during serial setup.

C
typedef enum {
  USART_PARITY_EVEN,
  USART_PARITY_ODD,
  USART_PARITY_NONE,
  USART_PARITY_INVALID
} USARTx_PARITY;
Members
Members 
Description 
USART_PARITY_EVEN 
Even Parity 
USART_PARITY_ODD 
Odd Parity 
USART_PARITY_NONE 
This enum is defined to set frame format only

  • This value won't be written to register

 

USART_PARITY_INVALID 
Force the compiler to reserve 32-bit memory for each enum 
Remarks

None.
