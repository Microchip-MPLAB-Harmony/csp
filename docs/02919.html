MCAN_MODE Enumeration

MCAN Mode 

This data type defines MCAN mode Classic CAN, CAN FD without BRS(Bit rate switching) and CAN FD with BRS.

C
typedef enum {
  MCAN_MODE_NORMAL = 0,
  MCAN_MODE_FD_WITHOUT_BRS,
  MCAN_MODE_FD_WITH_BRS
} MCAN_MODE;
Remarks

None.
