Library Interface
a) Initialization Function
 
Name 
Description 
 
SUPC_Initialize 
Initializes SUPC x module of the device 
b) Setup Functions
 
Name 
Description 
 
SUPC_CallbackRegister 
Allows application to register callback with PLIB. 
c) Transaction Functions
 
Name 
Description 
 
SUPC_GPBRRead 
Reads the data from General Purpose Backup Register(GPBR) 
 
SUPC_GPBRWrite 
Writes data to the General Purpose Backup Register(GPBR) 
 
SUPC_SleepModeEnter 
Puts the device into Sleep mode 
 
SUPC_WaitModeEnter 
Puts the device into Wait mode 
 
SUPC_BackupModeEnter 
Puts the device into Backup mode 
d) Data Types and Constants
 
Name 
Description 
 
SUPC_CALLBACK 
Pointer to a SUPC callback function 
 
WAITMODE_FLASH_STATE 
Identifies Flash Low-power mode 
 
WAITMODE_WKUP_SOURCE 
Identifies the wakeup input source
