RSTC_BKUPEXIT_CAUSE Type

Backup Exit Cause 

This typedef identifies whether the from Hibernate or RTC led to the backup reset.

C
typedef uint32_t RSTC_BKUPEXIT_CAUSE;
Remarks

Refer to the specific device data sheet to determine availability.

Macros
 
Name 
Description 
 
RSTC_BKUPEXIT_CAUSE_HIB0 
Backup reset due to Hibernate 
 
RSTC_BKUPEXIT_CAUSE_RTC 
Backup reset due to Real Timer Counter Interrupt
