USARTx_ERROR Enumeration

USART Errors 

This may be used to check the type of error occurred with the USART peripheral during error status.

C
typedef enum {
  USART_ERROR_NONE,
  USART_ERROR_PARITY,
  USART_ERROR_FRAMING,
  USART_ERROR_OVERRUN
} USARTx_ERROR;
Members
Members 
Description 
USART_ERROR_NONE 
Error status when no error has occurred 
USART_ERROR_PARITY 
Error status when parity error has occurred 
USART_ERROR_FRAMING 
Error status when framing error has occurred 
USART_ERROR_OVERRUN 
Error status when overrun error has occurred 
Remarks

None.
