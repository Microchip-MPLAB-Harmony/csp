CANx_MessageReceive Function

This routine receives a message from CAN bus.

C
bool CANx_MessageReceive(
    uint32_t * id, 
    uint8_t * length, 
    uint8_t * data, 
    uint32_t * timestamp, 
    uint8_t fifoNum
);
Preconditions

CANx_Initialize must have been called for the associated CAN instance.

Parameters
Parameters 
Description 
id 
Pointer to 11-bit / 29-bit identifier (ID) to be received. 
length 
Pointer to data length in number of bytes to be received. 
data 
Pointer to destination data buffer 
timestamp 
Pointer to Rx message timestamp, timestamp value is 0 if Timestamp is disabled in CFDxTSCON 
fifoNum 
FIFO number 
Returns

Request status. true - Request was successful. false - Request has failed.

Remarks

This routine receives a standard or extended messages from requested Rx FIFO.

Example
uint8_t rx_message[8], rx_messageLength;
uint32_t rx_messageID;
CAN1_Initialize();
CAN1_MessageReceive(&rx_messageID, &rx_messageLength, rx_message, 0, 2);
