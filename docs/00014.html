Introduction

The Analog Comparator peripheral (AC) supports two individual comparators. Each comparator (COMP) compares the voltage levels on two inputs (‘+’ and ‘-‘ input pins of the comparator), and provides a digital output based on this comparison. Each comparator may be configured to generate interrupt requests and/or peripheral events upon several different combinations of input change. 

The AC peripheral implements one pair of comparator. This pair can be set in window mode to compare a signal to a voltage range instead of a single voltage level.
