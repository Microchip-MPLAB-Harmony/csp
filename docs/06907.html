SQI_XIP_ADDR_BYTES Enumeration

SQI XIP Address Bytes can be used to specify the number of address bytes to be used while setting up the SQI peripheral in XIP mode.

C
typedef enum {
  ZERO_ADDR_BYTES = 0x0,
  ONE_ADDR_BYTES = 0x1,
  TWO_ADDR_BYTES = 0x2,
  THREE_ADDR_BYTES = 0x3,
  FOUR_ADDR_BYTES = 0x4
} SQI_XIP_ADDR_BYTES;
Remarks

None.
