RSWDT_MODE Enumeration

RSWDT Mode 

This may be used to initialize the RSWDT in window or normal mode.

C
typedef enum {
  RSWDT_MODE_NORMAL,
  RSWDT_MODE_WINDOW
} RSWDT_MODE;
Members
Members 
Description 
RSWDT_MODE_NORMAL 
Normal Mode of Operation 
RSWDT_MODE_WINDOW 
Window mode of operation 
Remarks

None.
