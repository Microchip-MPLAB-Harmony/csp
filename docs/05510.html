SERCOMx_SPI_Initialize Function

This function initializes instance x of SERCOM module operating in SPI mode. This function should be called before any other library function. The SERCOM module will be configured as per the MHC settings.

C
void SERCOMx_SPI_Initialize();
Preconditions

MCC GUI should be configured with the right values. The Generic Clock configuration and the SERCOM Peripheral Clock channel should have been configured in the clock manager GUI.The function will itself enable the required peripheral clock channel and main clock.

Returns

None.

Remarks

This function must be called once before any other SPI function is called.

Example
    SERCOM1_SPI_Initialize();
