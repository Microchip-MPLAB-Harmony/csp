Hardware Setup
  1. Project sam_9x60_ek.IAR
    • Hardware Used
      • SAM9X60 Evalutaion Kit (SAM9X60-EK)
      • SD Card with FAT32 file system
    • Hardware Setup
      • Setup the SD card
        • Download harmony MPU bootstrap loader from this location
        • Copy the downloaded boot loader binary( boot.bin) and generated application binary (harmony.bin)into the SD card
        • Insert the SD card into the SDMMC0 slot on the board
      • Connect the Debug USB port (J22) on the board to the computer using a micro USB cable
      • Use a jumper wire to short pin 32 of connector J17 with Pin RX of Mikrobus connector
      • Note: Reset push button on SAM9X60 EK is labeled as SW3

 

  1. Project sam_a5d2_xult.IAR OR sam_a5d2_xult_xc32.X
    • Hardware Used
      • SAM A5D2 Xplained Ultra board

    • Hardware Setup

        • Download harmony MPU bootstrap loader from this location
        • Copy the downloaded boot loader binary( boot.bin) and generated application binary (harmony.bin)into the SD card

      • Use a jumper wire to connect J8-PIN8 to J22-PIN6

 

  1. Project sam_c21n_xpro.X
    • Hardware Used
      • SAM C21N Xplained Pro Evaluation Kit.
    • Hardware Setup
      • TC0 generates the PWM waveform on pin PA09
      • TC1 is configured in capture mode. Capture trigger is provided by input event line. TC0 output is routed using EIC and EVSYS to TC1 input event line
      • Use a jumper wire to connect "Pin 4 of EXT2 (TC0_W01)" to "Pin 6 of EXT2 (EIC_EXTINT5)"
      • Connect the Debug USB port on the board to the computer using a micro USB cable
