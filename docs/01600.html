Running The Application
  1. Build and Program the application using the MPLAB X IDE.
  2. Observe the output as described in the following table:
Project Name 
Output 
sam_c21n_xpro.X 
Probe PORT pin PA02 (Pin DAC VOUT on the DAC Header on the SAM C21n Xplained Pro Evaluation Kit) and observe a sinusoidal waveform of 1 KHz.

To observe a sawtooth waveform of 1 KHz, press the switch SW0.
Subsequent pressing of the switch SW0 toggles the output between sine wave and sawtooth wave. 
sam_d11_xpro.X 
Probe PORT pin PA02 (Pin 3 of the EXT1 on the SAM D11 Xplained Pro Evaluation Kit) and observe a sinusoidal waveform of 1 KHz.

To observe a sawtooth waveform of 1 KHz, press the switch SW0.
Subsequent pressing of the switch SW0
