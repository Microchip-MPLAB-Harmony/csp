MCANx_InterruptClear Function
C
void MCANx_InterruptClear(
    MCAN_INTERRUPT_MASK interruptMask
);
Preconditions

MCANx_Initialize must have been called for the associated MCAN instance.

Parameters
Parameters 
Description 
interruptMask 
Interrupt to be cleared 
Returns

None

Example
MCAN_INTERRUPT_MASK interruptMask = MCAN_INTERRUPT_RF0W_MASK;
MCAN0_InterruptClear(interruptMask);
