ADC_ConversionStatusGet Function

This function returns the status of whether channel conversion is complete

C
bool ADC_ConversionStatusGet();
Preconditions

ADC_Initialize() function must have been called first for the associated instance. Also conversion must have been started.

Returns

Conversion status of the channel

  • False - Conversion is not finished or not started yet
  • True - Conversion is complete
Remarks

This function should be called only after conversion is started.

Example
  bool adcConversionStatus = false;
  ADC_Initialize();
  ADC_ConversionStart();
  adcConversionStatus = ADC_ConversionStatusGet();
