PORT_GROUP Enumeration

PORT Channel 

This enumeration identifies the port groups available on the device.

C
typedef enum {
  PORT_GROUP_A,
  PORT_GROUP_B,
  PORT_GROUP_C
} PORT_GROUP;
Members
Members 
Description 
PORT_GROUP_A 
Group A 
PORT_GROUP_B 
Group B 
PORT_GROUP_C 
Group C 
Remarks

The caller should not use the constant expressions assigned to any of the enumeration constants as these may vary between devices. 

Port group shown here are representative only. Not all ports groups are available on all devices. Refer to the specific device data sheet to determine which ports are supported. The MHC will generate this enumeration with the port groups that are available on the device.
