Library Interface
a) Initialization Function
 
Name 
Description 
 
RTC_Initialize 
Initialize given instance of the RTC peripheral. 
b) Setup Functions
 
Name 
Description 
 
RTC_Timer32CallbackRegister 
Register the callback function to be called when an 32-bit Timer Interrupt occurs. 
c) Transaction Functions
 
Name 
Description 
 
RTC_Timer32Start 
Starts the 32-bit timer. 
 
RTC_Timer32Stop 
Stops the 32-bit timer from counting. 
 
RTC_Timer32CounterSet 
Set the 32-bit Timer Counter Value. 
 
RTC_Timer32CounterGet 
Get the current 32-bit counter value. 
 
RTC_Timer32CompareSet 
Set the 32-bit timer period value. 
 
RTC_Timer32PeriodGet 
Get 32-bit timer period Value. 
 
RTC_Timer32FrequencyGet 
Returns the frequency at which the 32-bit timer counter is operating. 
 
RTC_Timer32InterruptDisable 
Disable selected RTC interrupt 
 
RTC_Timer32InterruptEnable 
Enable selected RTC interrupt 
d) Status Functions
 
Name 
Description 
 
RTC_Timer32CompareHasMatched 
Check for 32-bit Timer Compare match 
 
RTC_Timer32CounterHasOverflowed 
Check if the 32-bit counter overflow. 
e) Data Types and Constants
 
Name 
Description 
 
RTC_TIMER32_CALLBACK 
Defines the data type and function signature of the RTC 32-bit Timer Counter callback function. 
 
RTC_TIMER32_INT_MASK 
Possible RTC 32-bit Timer Counter Mode Events.
