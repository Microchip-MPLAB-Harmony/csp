CANx_MessageReceive Function
C
bool CANx_MessageReceive(
    uint32_t * id, 
    uint8_t * length, 
    uint8_t * data, 
    CAN_MAILBOX_RX_ATTRIBUTE mailboxAttr
);
Preconditions

CANx_Initialize must have been called for the associated CAN instance.

Parameters
Parameters 
Description 
id 
Pointer to 11-bit / 29-bit identifier (ID) to be received. 
length 
Pointer to data length in number of bytes to be received. 
data 
pointer to destination data buffer 
mailboxAttr 
Mailbox type either RX Mailbox or RX Mailbox with overwrite 
Returns

Request status. true - Request was successful. false - Request has failed.

Remarks

This routine receives a standard or extended messages based upon the CAN Plib setup.

Example
uint8_t buf[8], id, length;
CAN0_Initialize();
CAN0_MessageReceive(&id, &length, buf, CAN_MAILBOX_DATA_FRAME_RX_OVERWRITE);
