I2C_SLAVE_TRANSFER_EVENT Enumeration

I2C Slave Transfer Event Enums. 

Defines the enum for the I2C slave transfer event. An event of this type is returned back in the callback by the I2C slave PLIB to indicate the transfer event for which the callback has been called.

C
typedef enum {
  I2C_SLAVE_TRANSFER_EVENT_NONE = 0,
  I2C_SLAVE_TRANSFER_EVENT_ADDR_MATCH,
  I2C_SLAVE_TRANSFER_EVENT_RX_READY,
  I2C_SLAVE_TRANSFER_EVENT_TX_READY,
  I2C_SLAVE_TRANSFER_EVENT_ERROR
} I2C_SLAVE_TRANSFER_EVENT;
