Using the Library

The Low Power Asynchronous Receiver (RXLP) is a low-power UART with a slow clock. It works only in Receive mode. It features a Receive Data (RXD) pin that can be used to wake up the system. 

 

System wake-up: 

void switchHandler( PIO_PIN pin, uintptr_t context) 

{ 

SHDWC_Shutdown(); 

} 

 

int main ( void ) 

{ 

/* Initialize all modules */ 

SYS_Initialize ( NULL ); 

PIO_PinInterruptCallbackRegister(SWITCH_PIN, switchHandler, 0); 

PIO_PinInterruptEnable(SWITCH_PIN); 

while ( true ) 

{ 

PIT_DelayMs(500); 

LED_Toggle(); 

/* Maintain state machines of all polled MPLAB Harmony modules. */ 

SYS_Tasks ( ); 

} 

/* Execution should not come here during normal operation */ 

return ( EXIT_FAILURE ); 

}
