NVMCTRL_Read Function

Reads length number of bytes from a given address in FLASH memory into the user buffer.

C
bool NVMCTRL_Read(
    uint32_t * data, 
    uint32_t length, 
    uint32_t address
);
Preconditions

None

Returns

None.

Example
    uint8_t buffer[256];
    NVMCTRL_Read( (uint32_t *) buffer, 256, 0x20000);
