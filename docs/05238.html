SDHC_RESET_TYPE Enumeration

SDHC error reset type 

The enumeration is used to select the error type to reset.

C
typedef enum {
  SDHC_RESET_ALL,
  SDHC_RESET_CMD,
  SDHC_RESET_DAT
} SDHC_RESET_TYPE;
Members
Members 
Description 
SDHC_RESET_ALL 
Reset all errors 
SDHC_RESET_CMD 
Reset command errors 
SDHC_RESET_DAT 
Reset data errors 
Remarks

None.
