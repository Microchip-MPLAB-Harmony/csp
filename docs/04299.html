Running The Application

Running project with MPLAB X IDE 

  1. Build and Program the application using the MPLAB X IDE
  2. Observe the high-side and low-side PWM waveforms on the oscilloscope. Observe the dead time between the high side and low side.
    • PWM Channel 0
      • Pin 7 of EXT1 - PWM0_PWMH0 (PA00)
      • Pin 4 of EXT1 - PWM0_PWML0 (PA19)
    • PWM Channel 1
      • Pin 9 of EXT2 - PWM0_PWMH1 (PA02)
      • Pin 15 of EXT1 - PWM0_PWML1 (PD25)
    • PWM Channel 2
      • Pin 7 of EXT2 - PWM0_PWMH2 (PC19)
      • Pin 8 of EXT2 - PWM0_PWML2 (PD26)
  3. Observe the output as described in the following table:
Project Name 
Output 
sam_e70_xult.X 
Success: Observe the high side and low side PWM waveforms. PWM frequency = 20KHz.
Duty cycle of the waveform is changing from 0% to 100%. 
sam_v71_xult.X 
Success: Observe the high side and low side PWM waveforms. PWM frequency = 20KHz.
Duty cycle of the waveform is changing from 0% to 100%. 

 

 

Running project with IAR Embedded work bench

  1. Build the project using IAR Embedded Workbench
  2. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  3. Reset the board to run the application

 

Project Name 
Output 
sam_a5d2_xult.IAR 

  • Observe the High side of PWM Channel 2 on pin 7 of XPRO_EXT2 (J26)
  • PWM frequency is 20 KHz
  • PWM duty cycle varies from 0 to 100 %
