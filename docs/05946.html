Library Interface
a) Initialization Function
 
Name 
Description 
 
TCx_CHy_CompareInitialize 
Initializes given TC channel in compare mode. 
b) Setup Functions
 
Name 
Description 
 
TCx_CHy_CompareCallbackRegister 
Registers the function to be called from interrupt. 
c) Transaction Functions
 
Name 
Description 
 
TCx_CHy_CompareStart 
Starts the given TC channel compare counter. 
 
TCx_CHy_CompareStop 
Stops the given TC channel counter in compare mode. 
 
TCx_CHy_ComparePeriodSet 
Sets the period value of a given TC channel in compare mode. 
 
TCx_CHy_ComparePeriodGet 
Gets the period value of given timer channel in compare mode. 
 
TCx_CHy_CompareFrequencyGet 
Provides the given timer's counter-increment frequency in compare mode. 
 
TCx_CHy_CompareASet 
Sets the Compare-A value of the given timer channel in compare mode. 
 
TCx_CHy_CompareBSet 
Sets the Compare-B value of the given timer channel in compare mode. 
d) Status Functions
 
Name 
Description 
 
TCx_CHy_CompareStatusGet 
Identifies status of the compare events 
e) Data Types and Constants
 
Name 
Description 
 
TCx_COMPARE_CALLBACK 
Defines the function pointer data type and function signature for the tc channel callback function. 
 
TCx_COMPARE_CALLBACK_OBJECT 
Callback structure 
 
TCx_COMPARE_STATUS 
Identifies channel interrupt source mask
