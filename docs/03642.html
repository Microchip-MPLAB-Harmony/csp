HSMCI_BlockSizeSet Function

Sets the size of the one data block of transfer.

C
void HSMCI_BlockSizeSet(
    uint16_t blockSize
);
Preconditions

HSMCI_Initialize() must have been called first.

Parameters
Parameters 
Description 
blockSize 
Size of one block of transfer. 
Returns

None.

Remarks

None.

Example
HSMCI_BlockSizeSet(512);
