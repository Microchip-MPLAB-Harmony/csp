Hardware Setup
  1. Project: sam_e70_xult.X
    • Hardware Used
      • SAM E70 Xplained Ultra Evaluation Kit
    • Hardware Setup
      • Connect the DACC "Channel 0" and "Channel 1" output pin observable at PB13 (Pin 5 of the J502) and PD00 (Pin 6 of the J502) respectively to oscilloscope probe, to observe the analog output waveform
      • Connect the Debug USB port on the board to the computer using a micro USB cable
