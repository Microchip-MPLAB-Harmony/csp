Library Interface
a) Initialization Function
 
Name 
Description 
 
UARTx_Initialize 
Initializes given instance of the UART peripheral. 
b) Setup Functions
 
Name 
Description 
 
UARTx_SerialSetup 
Sets the UART serial communication settings dynamically. 
 
UARTx_ReadCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given UART's read events occur. 
 
UARTx_WriteCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given UART's write events occur. 
c) Transaction Functions
 
Name 
Description 
 
UARTx_Read 
Submits request to read n-Bytes of data to the given UART peripheral. 
 
UARTx_Write 
Submits a write buffer to the given UART peripheral to transfer. 
 
UARTx_ReadByte 
Submits request to read a byte of data to the given UART peripheral. 
 
UARTx_WriteByte 
Submits a byte of data to the given UART peripheral to transfer. 
d) Status Functions
 
Name 
Description 
 
UARTx_ErrorGet 
Gets the error of the given UART peripheral instance. 
 
UARTx_ReadCountGet 
Returns the count of number of bytes processed for a given UART read operation. 
 
UARTx_WriteCountGet 
Returns the count of number of bytes processed for a given UART write operation. 
 
UARTx_ReadIsBusy 
Returns the read request status associated with the given UART peripheral instance. 
 
UARTx_WriteIsBusy 
Returns the write request status associated with the given UART peripheral instance. 
 
UARTx_ReceiverIsReady 
Returns the hardware status of the UART Receiver. 
 
UARTx_TransmitterIsReady 
Returns the hardware status of the UART Transmitter. 
e) Data Types and Constants
 
Name 
Description 
 
UART_CALLBACK 
Defines the data type and function signature for the UART peripheral callback function. 
 
UART_DATA 
Defines the data type for the UART data bit length. 
 
UART_ERROR 
Defines the data type for the UART peripheral errors. 
 
UART_PARITY 
Defines the type of parity for UART peripheral. 
 
UART_SERIAL_SETUP 
Defines the data type for the UART serial setup. 
 
UART_STOP 
Defines the length of stop bits.
