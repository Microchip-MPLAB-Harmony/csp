MPLAB Harmony Configurations

Refer to the MHC project graph for the components used and the respective configuration options. 

 

The FREQM peripheral is configured to count the number of periods of the measurement clock over 128 cycles of the reference clock. 

The following clock configuration is selected to measure the frequency of the 48MHz internal oscillator using 32-kHz Crystal oscillator as the reference clock.

  • The 48MHz internal oscillator is selected as the clock source for GCLK0. The GCLK0 is used as the measurement clock for the FREQM peripheral.
  • The 32KHz crystal oscillator is selected as the clock source for GCLK1. The CLK1 is used as the reference clock for the FREQM peripheral.
