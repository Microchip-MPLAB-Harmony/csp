WAITMODE_FLASH_STATE Enumeration

Flash Low-power mode states 

This enumeration identifies Flash Low-power mode state when system enters Wait mode.

C
typedef enum {
  WAITMODE_FLASH_STANDBY,
  WAITMODE_FLASH_DEEPSLEEP
} WAITMODE_FLASH_STATE;
Members
Members 
Description 
WAITMODE_FLASH_STANDBY 
Flash is in Standby mode 
WAITMODE_FLASH_DEEPSLEEP 
Flash is in Deep-powerdown mode 
Remarks

None.
