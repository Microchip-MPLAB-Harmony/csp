ACC_STATUS_SOURCE Enumeration

ACC comparison status 

This enumeration identifies the ACC Comparison status either from Comparison edge or synchronized comparator output.

C
typedef enum {
  ACC_STATUS_SOURCE_COMPARATOR_OUTPUT,
  ACC_STATUS_SOURCE_COMPARISON_EDGE
} ACC_STATUS_SOURCE;
Members
Members 
Description 
ACC_STATUS_SOURCE_COMPARATOR_OUTPUT 
ACC synchronized comparator output 
ACC_STATUS_SOURCE_COMPARISON_EDGE 
ACC comparison edge 
Remarks

None.
