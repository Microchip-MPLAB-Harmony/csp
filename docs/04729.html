Running The Application

MCU Devices 

  1. Build and program the application using the MPLAB X IDE.
  2. The LED is turned ON when the switch is pressed and turned OFF when the switch is released.

The following table provides the switch and the LED name.

MPU Devices

  1. Build the IAR or MPLAB X project using IAR Embedded Workbench or MPLAB X IDE respectively
  2. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  3. Reset the board to run the application
  4. The LED is turned ON when the switch is pressed and turned OFF when the switch is released
