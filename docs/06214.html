TWIHSx_IsBusy Function

This function returns the TWIHS Peripheral busy status

C
bool TWIHSx_IsBusy();
Preconditions

TWIHSx_Initialize must have been called for the associated TWIHS instance.

Returns
  • True - Busy.
  • False - Not busy.
Remarks

None.

Example
if(!TWIHS1_IsBusy())
{
    // add another transfer request
}
