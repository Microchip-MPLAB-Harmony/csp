TMRx_CounterGet Function

This function reads the timer counter value.

C
uint16_t TMRx_CounterGet();
Preconditions

TMRx_Initialize() function must have been called first.

Returns

counter - The timer's current count value.

Remarks

If timer is operating on 32 bit mode then return value should be uint32_t type.

Example
uint16_t counter;

TMR2_Initialize();
TMR2_Start();
counter = TMR2_CounterGet();
