Library Interface
a) Initialization Function
 
Name 
Description 
 
DMAC_Initialize 
This function initializes the DMAC controller of the device. 
b) Setup Functions
 
Name 
Description 
 
DMAC_ChannelCallbackRegister 
This function allows a DMAC PLIB client to set an event handler. 
 
DMAC_ChannelDisable 
This function disables the DMA channel. 
c) Transaction Functions
 
Name 
Description 
 
DMAC_ChannelTransfer 
Schedules a DMA transfer on the specified DMA channel. 
d) Status Functions
 
Name 
Description 
 
DMAC_ChannelIsBusy 
Reads the busy status of a channel. 
e) Data Types and Constants
 
Name 
Description 
 
DMAC_CHANNEL 
Fundamental data object that represents DMA channel number. 
 
DMAC_CHANNEL_CALLBACK 
Pointer to a DMAC Transfer Event handler function. 
 
DMAC_TRANSFER_EVENT 
Identifies the available DMA operating modes.
