WDT_Enable Function

This function enables the WDT peripheral. Calling this function will cause the WDT to start counting up to the configured timeout value. This function also enables the Early Warning interrupt if this interrupt has been enabled in MHC.

C
void WDT_Enable();
Preconditions

None.

Returns

None.

Remarks

None.

Example
WDT_Enable();
