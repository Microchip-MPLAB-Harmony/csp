TRNGx_RandomNumberGenerate Function

This function triggers the TRNG peripheral to generate a 32-bit random number. The API will return in case of interrupt mode and wait for random number generation to complete in case of polled mode.

C
void TRNGx_RandomNumberGenerate();
Preconditions

None.

Returns

None.

Remarks

This API should be used when the library is configured in interrupt mode.

Example
    TRNG0_RandomNumberGenerate();
