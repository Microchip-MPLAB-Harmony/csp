Library Interface
a) Initialization Function
 
Name 
Description 
 
PWMx_Initialize 
Initializes given instance of PWM peripheral 
b) Setup Functions
 
Name 
Description 
 
PWMx_CallbackRegister 
Registers the function to be called from interrupt 
c) Transaction Functions
 
Name 
Description 
 
PWMx_ChannelCounterEventDisable 
Disables counter event of given channels 
 
PWMx_ChannelCounterEventEnable 
Enables counter event of given channels 
 
PWMx_ChannelDutySet 
Writes the duty cycle value of given PWM channel 
 
PWMx_ChannelPeriodGet 
Reads the period value of given PWM channel 
 
PWMx_ChannelPeriodSet 
Sets the period value of given PWM channel 
 
PWMx_ChannelsStart 
Starts the given PWM channels 
 
PWMx_ChannelsStop 
Stops the given PWM channels 
d) Status Functions
 
Name 
Description 
 
PWMx_ChannelCounterEventStatusGet 
Disables counter event of given channels 
e) Data Types and Constants
 
Name 
Description 
 
PWM_CALLBACK 
Defines the data type and function signature for the PWM peripheral callback function. 
 
PWM_CALLBACK_OBJECT 
Callback structure 
 
PWM_CHANNEL_MASK 
Identifies PWM channel mask 
 
PWM_CHANNEL_NUM 
Identifies PWM channel number in a PWM peripheral
