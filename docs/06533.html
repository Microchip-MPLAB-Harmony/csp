Running The Application

Running project with MPLAB X IDE 

  1. Open the Terminal application (Ex.:Tera term) on the computer.
  2. Connect to the EDBG Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build and Program the application using the MPLAB X IDE.
  4. Console displays the following message:

  1. Follow the below table for switch deatils,

  1. Press the on-board switch to generate a 32-bit random number.

 

Running project with IAR Embedded work bench

  1. Connect to the EDBG Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  2. Build the project using IAR Embedded Workbench
  3. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  4. Perform a warm reset of the board (short press and release of the reset push button) to run the application
  5. Console displays the following message:

  1. Press the on-board switch to generate a 32-bit random number.

NOTE: In addition to the application, boot loader also uses the console to print its log messages. Since these boot loader messages are not relevant to the application under test, it is not documented here.
