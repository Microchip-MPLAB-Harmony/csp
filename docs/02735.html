HSMCI_DATA_ERROR Macro

HSMCI data erorr 

This macro is a bit wise OR of all the possible errors corresponding to a data transfer. This may be used by the client to test if the data transfer completed with or without errors by ANDing this macro with the return value of the HSMCI_DataErrorGet() function.

C
#define HSMCI_DATA_ERROR (HSMCI_SR_UNRE_Msk | HSMCI_SR_OVRE_Msk | HSMCI_SR_DTOE_Msk | HSMCI_SR_DCRCE_Msk)
Remarks

None.
