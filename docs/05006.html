RTC_Timer16InterruptDisable Function

This function allows the application to disable period match, compare match and periodic event interrupt

C
void RTC_Timer16InterruptDisable(
    RTC_TIMER16_INT_MASK interrupt
);
Preconditions

RTC_Initialize must have been called for the associated RTC instance. The Interrupt option in MHC should have been enabled. The RTC peripheral should have been configured for 16-bit timer counter operation. A valid event handling callback function should have been registered by calling the RTC_Timer16CallbackRegister() function.

Parameters
Parameters 
Description 
InterruptMask 
Specifies interrupt source to be disabled 
Returns

None.

Remarks

None.

Example
RTC_Timer16InterruptDisable(RTC_TIMER16_INT_MASK_PERIOD_MATCH);
