EVSYS_InterruptEnable Function

This function enables the specified evsys interrupt.

C
void EVSYS_InterruptEnable(
    EVSYS_INT_MASK interrupt
);
Preconditions

EVSYSx module should be initialized with the required configuration parameters from the MHC GUI in the EVSYS_Initialize() function

Parameters
Parameters 
Description 
interrupt 
Interrupt to be enabled 
Returns

None

Remarks

Interrupts configuration will be generated in EVSYS_Initialize() function as per user settings in MHC. The EVSYS_InterruptEnable and EVSYS_InterruptDisable API should be used to enable and disable interrupt at run time.

Example
    // Enable Event System channel 2 Event Detection interrupt 
    EVSYS_InterruptEnable(EVSYS_INTENSET_EVD2);
