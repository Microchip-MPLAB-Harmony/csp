Running The Application

MCU Devices 

  1. Open the Terminal application (Ex.:Tera term) on the computer.
  2. Connect to the EDBG Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build and Program the application using the MPLAB X IDE.
  4. The console displays the following message

 

  1. The watch timer is fed periodically using System Timer (SysTick) to prevent the WDT reset and the LED is toggled.

MPU Devices 

  1. Connect to the EDBG Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  2. Build the IAR or MPLAB X project using IAR Embedded Workbench or MPLAB X IDE respectively
  3. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  4. The console displays the following message
