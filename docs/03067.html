Running The Application

IAR Embedded work bench

  1. Open the Terminal application (Ex.:Tera term) on the computer
  2. Connect to the DEBUG UART Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build the project using IAR Embedded Workbench
  4. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  5. Execute the steps and observe the output as described in the following table

  • CAN message will be displayed in the terminal window
  • RGB LED will toggle in green color for every message received
  • Received message will be echoed back to the CAN analyzer software and terminal window.
