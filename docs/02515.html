Library Interface
a) Initialization Function
 
Name 
Description 
 
HSMCI_Initialize 
Initializes the HSMCI peripheral and the internal data structures used by the peripheral library. 
 
HSMCI_ModuleInit 
Initializes the HSMCI peripheral. 
b) Setup Functions
 
Name 
Description 
 
HSMCI_BlockCountSet 
Sets the number of blocks to transfer. 
 
HSMCI_BlockSizeSet 
Sets the size of the one data block of transfer. 
 
HSMCI_BusWidthSet 
Configures the width of the data bus. 
 
HSMCI_CallbackRegister 
This function allows a HSMCI PLIB client to set an event handler. 
 
HSMCI_ClockSet 
Sets the High Speed MultiMedia Card Interface clock frequency. 
 
HSMCI_SpeedModeSet 
Sets the bus speed. 
c) Transaction Functions
 
Name 
Description 
 
HSMCI_ResponseRead 
Reads the response to a given command. 
 
HSMCI_CommandSend 
This function allows the client to send a SD command on the HSMCI interface. 
 
HSMCI_DmaSetup 
Sets up the DMA for data tranfers. 
d) Status Functions
 
Name 
Description 
 
HSMCI_CommandErrorGet 
Returns the errors associated with a command transfer. 
 
HSMCI_DataErrorGet 
Returns the errors associated with a data transfer. 
 
HSMCI_IsCmdLineBusy 
Returns the status of the command line. 
 
HSMCI_IsDatLineBusy 
Returns the status of the data line. 
e) Data Types and Constants
 
Name 
Description 
 
HSMCI_BUS_WIDTH 
The enumeration lists the bus widths for the HSMCI bus. 
 
HSMCI_CALLBACK 
The prototype of the HSMCI callback function 
 
HSMCI_CMD_RESP_TYPE 
The enumeration lists the supported response types 
 
HSMCI_DATA_TRANSFER_DIR 
The enumeration lists the direction of the data transfer. 
 
HSMCI_DATA_TRANSFER_TYPE 
The enumeration lists single or multi block data transfer type. 
 
HSMCI_DataTransferFlags 
The data structure is used by the client to provide information about the data trasnfer to the HSMCI PLIB. 
 
HSMCI_ERROR_FLAGS 
The enumeration lists the possible error types for a command and data transfer. 
 
HSMCI_READ_RESPONSE_REG 
The enumeration lists the response registers that can be read for a given command. 
 
HSMCI_SPEED_MODE 
The enumeration lists the bus widths for the HSMCI bus. 
 
HSMCI_XFER_STATUS 
The enumeration lists the status of the transfer. 
 
HSMCI_CLOCK_FREQ_400_KHZ 
This is macro HSMCI_CLOCK_FREQ_400_KHZ. 
 
HSMCI_CLOCK_FREQ_DS_25_MHZ 
This is macro HSMCI_CLOCK_FREQ_DS_25_MHZ. 
 
HSMCI_CLOCK_FREQ_HS_50_MHZ 
This is macro HSMCI_CLOCK_FREQ_HS_50_MHZ. 
 
HSMCI_CMD_ERROR 
The macro is a bit wise OR of all the possible errors corresponding to a command transfer. 
 
HSMCI_DATA_ERROR 
The macro is a bit wise OR of all the possible errors corresponding to a data transfer.
