Hardware Setup
  1. Project sam_9x60_ek.IAR
  • Hardware Used
    • SAM 9X60 Evaluation Kit
    • SD Card with FAT32 file system
  • Hardware Setup
    • Setup the SD card
      • Download harmony MPU bootstrap loader from this location
      • Copy the downloaded boot loader binary( boot.bin) and generated application binary (harmony.bin)into the SD card
      • Insert the SD card into the SDMMC0 slot on the board
    • Connect the Debug USB (J22) port on the board to the computer using a micro USB cable
    • Connect PB11 (PIN 32 of Connector J17) to an an oscilloscope/Logic analyzer
    • Connect PB12 (PIN 33 of Connector J17) to an an oscilloscope/Logic analyzer
    • Connect PB13 (PIN 31 of Connector J17) to an an oscilloscope/Logic analyzer
    • Note: Reset push button on SAM9X60 EK is labeled as SW3
