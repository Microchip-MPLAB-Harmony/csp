sdadc_sample

In this application, the DAC output is fed to an analog input for the SDADC conversion. An analog input is converted by a software trigger and the converted value is displayed on the console. 

The DAC output is changed by 0.1 V upon switch press. 

 

The SDADC sample output is between 0.0 V to 3.3 V and it starts from 1.65 V when device is reset.
