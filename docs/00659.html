ADCHS_CALLBACK Type

Callback Function Pointer 

This data type defines the function pointer and function signature for the adchs peripheral callback function. The adchs peripheral will call back the client's function with this signature at the end of conversion.

C
typedef void (* ADCHS_CALLBACK)(ADCHS_CHANNEL_NUM channel, uintptr_t context);
Preconditions

ADCHS_Initialize must have been called for the given adchs peripheral instance and ADCHS_CallbackRegister must have been called to set the function to be called for respective analog channel.

Parameters
Parameters 
Description 
channel 
Analog channel number 
context 
Allows the caller to provide a context value (usually a pointer to the callers context for multi-instance clients). 
Returns

None.

Remarks

None.

Example
void ADCHS_ChCallback (ADCHS_CHANNEL_NUM channel, uintptr_t context);

ADCHS_CallbackRegister(ADCHS_ChCallback, NULL);
